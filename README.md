# Comprehensive Hospital Analytics Dashboard (Power BI)

This project focuses on healthcare analytics using Power BI to evaluate
hospital performance, patient trends, and financial metrics.

## Dataset Overview
- Total Visits Analyzed: 16,000+
- Total Patients: 2,000
- Total Billing: ₹25M
- Insurance Revenue: ₹5M

## Dashboard Highlights
- KPI tracking and performance metrics
- Billing trend analysis
- Diagnosis category distribution
- Department and specialization insights
- Payment mode analysis

## Tools Used
Power BI • Data Modeling • DAX • Data Visualization
